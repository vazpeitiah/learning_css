Flex Box

Información extraida de Mozilla Developer Page

Flex box es un método de diseño de página unidimensional para compaginar elementos en filas o columnas. Los elementos de contenido crecen o encogen para rellenar los espacios del contenedor.

Para comenzar a utilizar caja flexibles vamos a utilizar la propiedad display: flex;, en el contenedor que queremos.

El modelos flexible

Cuando los elementos se presentan como cajas flexibles, se distribuyen con respecto a dos ejes:

  • El main axis es el eje que corre en la dirección en la que se colocan los elementos flexibles.
  • El cross axis es el eje que corre perpedicular a la dirección en la que se colocan los elementos flexibles.
  • El elementos padre flex container es al que le aplicamos la propiedad display: flex;.
  • Los elementos que se presentan como cajas flexibles dentro de flex container se denominan elementos flexibles.

Columnas y filas

Para establecer la dirección que tendran los elementos flexibles se utiliza la propiedad flex-direction. Por defecto, esta tiene un valor de row, pero podemos establecer el valor column para que los elementos flexibles se muestren en una columna.
