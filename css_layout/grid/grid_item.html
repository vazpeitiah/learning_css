Grid item

Un contenedor de cuadricula contiene items de cuadricula. Por defecto, un contenedor tiene una grilla de 1X1, pero podemos ir agregando multiples items como sea necesario.

Las propiedades grid-column y grid-row

Esta propiedad define cuantas columnas o filas va a ocupar un item en la grilla. Por ejemplo, el item-1, ocupa 4 columnas, y el item-2 ocupa 3 filas:
