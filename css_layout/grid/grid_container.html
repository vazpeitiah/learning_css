Grid container

Para hacer que un elemento HTML se comporte como una cuadricula(grid), debemos usar la propiedad display con el valor grid o inline-grid.

Después debemos de utilizar la propiedad grid-template-columns o grid-template-rows para definir el tamaño de nuestras filas y columnas. Podemos usar el valor auto, si queremos que las columnas o filas sean del mismo tamaño o también podemos usar un tamaño en especifico, en pixeles por ejemplo.
